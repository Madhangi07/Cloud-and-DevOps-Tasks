# Exercise 24 - DynamoDB Application Deployment

## Objective

Build a Python application that performs CRUD operations on AWS DynamoDB.

## Technologies

- Python 3.10
- AWS CLI
- Boto3
- DynamoDB

## Features

- Create Customer
- Read Customer
- Update Customer
- Delete Customer

## Run

```bash
pip install -r requirements.txt
python app.py
```

## AWS Table

Customer

Partition Key

CustomerId (String)
